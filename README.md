### Features

#### Streaming & UI
* **Triple Server Support:** Switch between S1, S2, and S3 for stable playback.
* **Sub & Dub:** Available for most titles.
* **Title Toggle:** Dedicated button to switch between English and Romaji titles.
* **Episode Lists:** Fast navigation through all available episodes.

#### User System
* **Local Watchlist:** Save favorites directly in the browser via local storage.
* **Mobile APK:** Native Android version for mobile viewing.
* **Data Management:** APK saves data to `/downloads/slothanimes_data`.
