Features
Streaming & Interface
 * Triple Server Support: Switch between S1, S2, and S3.
 * Sub & Dub: Available for most titles.
 * Title Toggle: Button to switch between English and Romaji titles.
 * Episode Lists: Fast navigation through available episodes.
User System
 * Local Watchlist: Save favorites directly in the browser via local storage.
 * Mobile APK: Native Android version for mobile viewing.
 * Data Management: APK saves data to /downloads/slothanimes_data.
